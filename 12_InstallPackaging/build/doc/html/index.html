TESTS FOR STACK BUFFER Documentation

This program is controlled by the TEST_COMMAND environment variable, initialized before compiling the program with a value from 0 to 21, to select a specific test. This program is initialized with the value 0.

The following tests are available:

  • 0 - Benchmark.
  • 1 - Buffer volume test.
  • 2 - Buffer size test.
  • 3 - Test for the buffer size after making an element.
  • 4 - Test for the correct value of this element.
  • 5 - Buffer cleaning test.
  • 6 - Test whether the buffer is not released.
  • 7 - Test for freeing up memory for the buffer.
  • 8 - Test for the size of the freed buffer.
  • 9 - Test for the value of the pointer of the freed buffer.
  • 10 - Size test for 10,000 items.
  • 11 - Test for the correct element search.
  • 12 - Buffer size increase test.
  • 13 - Memory size test after buffer grow.
  • 14 - Buffer Truncation test.
  • 15 - Tests for adding and removing using push buffer.
  • 16 - Tests for adding and removing using pop buffer.
  • 17-21 - Tests for adding and removing using push and pop buffers.
